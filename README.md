# 🩸 Sugar Reminder
Простая программа для автоматического напоминания об измерении сахара в крови.

## ✨ Функции
*   **Автозапуск**: Программа запускается сама при включении ПК.
*   **Таймер**: Настройка уведомлений каждые 30 мин, 1 час или 2 часа через меню в трее.
*   **Уведомление**: Появляется по центру экрана со звуковым сигналом.
*   **Стильный дизайн**: Черный фон с белым светящимся текстом.

## 📥 Как установить
1. Перейдите в раздел **Releases** справа.
2. Скачайте файл `sugar-reminder Setup 2.0.0.exe`.
3. Запустите установщик и следуйте инструкциям.

## 🛠 Технологии
* Node.js
* Electron
